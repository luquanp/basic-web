SmartGit, GitHub和Angular的环境搭建

  1. 安装NODEJS
  2. 验证NODEJS是否安装成功
    3. npm -v
  3. 安装angular脚手架
    4. npm install -g @angular/cli
  4. 验证angular是否安装成功
    5. ng -v
  5. 创建一个前台项目
    1. 新建一个文件夹D:\basic-web
    2. 开始-》运行-》cmd
      3.         d:
 	cd basic-web
    3. 创建一个angular前台项目
      4. ng new basic-web
    4. 安装所需的依赖插件
      5. ng install
    5. 启动前台项目
      6. ng serve --open
  6. 建立git本地库 git-repo
    1. git创建账号
    2. 邮箱激活账户
    3. 创建私钥
      4. 参照： 生成一个新的SSH KEY以及如何添加到SSH AGENT。
      • 生成一个新的SSH KEY
        1. 打开终端。
        2. 粘贴下面的命令行，注意使用注册GitHub用的邮箱地址
          3. ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

          这样就会生成一个新SSH KEY，邮箱地址会被当成一个标签。并且显示下面的文字：
          Generating public/private rsa key pair.
    4. 在github上填写公钥
      5. 参照： 如何添加SSH KEY到GitHub帐户上
